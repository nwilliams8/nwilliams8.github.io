Projects

Circular Motion

This project is a generative art piece scripted in JavaScript using the p5js which allowed me to have a bigger variety of shapes and functions to use when developing the project. I wanted this project to explore the sin and cos functions and give the user the oppertunity to watch as these functions developed their own unique pattern across the page with a small level of interaction where the user could change the color, speed and angle of the shapes.

Data Visualisation

This project is a data Visualisation on all the stage 1 marvel films using an online movie database. The intent of undertaking this project was to develop my skill in converting data from figures into something more visual and if I had more time I wouldve added more content and better visuals.

Video Piracy

This project was made to demonstrate the impact of piracy on the movie industry and how it is a product of modern technology. I wanted to demonstrate this using an old video cassette as i feel that it holds a strong nostalgia for many people, it also emphasises the speed that technology is moving at.

Digital Signals

This project was made using a copper coil attatched to a microphone, the intent was to capture the signals going on all around us since technology is a part of our everyday lives. The signals captured in this piece are from different components of a computer.
