City Interactive
Previous Next

This project was made as part of the DAT505 module, for it I designed I designed a small section of a city with different buildings and styles. This project was challenging as whilst I enjoyed the modelling aspect I needed to be able to get it to render using three.js which proved challenging and I had to comprimise the textures in order to allow the project to be rendered. I decided to use simple, bold colours to make each aspect of this project stand out. When it came to the interaction I started by adding the time of day which the user could alter using the DAT.gui menu, this was a good start as it was simple and the users could see the impact it had on the city straight away. After that I decided to add something a bit more complex to the scene which was weather effects, due to the timescale I only had time to add in rain and wind. The rain was added using a loop function in the render part of the scene where after the particles where generated they were reset to the bottom of the cloud model, to impliment the wind I added a horizontal direction to the rain which could be changed using the DAT.gui slider.
